布局技巧

两行布局：

  1. 使用 flex-wrap: wrap 让标签自动换行
    设置 max-height: 48px 限制为两行高度
    超出省略：
    使用 overflow: hidden 隐藏超出部分
    当标签数量超过6个时自动添加 overflow class
  2. 通过 ::after 伪元素在右下角显示省略号
    视觉效果：
    省略号使用渐变背景，从透明到白色，视觉过渡更自然
    省略号颜色为 #999，与其他次要文本保持一致
    CSS 样式说明：
    max-height: 48px: 两行标签的高度 (每行22px + 4px间距)
    flex-wrap: wrap: 允许标签换行
    gap: 4px: 标签之间的间距
    渐变背景: 让省略号看起来更自然地覆盖在标签上

动态判断逻辑：

overflow
当标签数量超过6个时，会添加 overflow class，显示省略号。您可以根据实际标签大小调整这个数量阈值。
现在标签容器支持两行显示，超出两行时会显示省略号，既保持了界面整洁，又提供了内容溢出的视觉提示。
