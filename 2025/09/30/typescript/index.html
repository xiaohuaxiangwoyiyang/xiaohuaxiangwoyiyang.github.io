1. ! 非空断言

感叹号 ! 的含义：

非空断言操作符：告诉 TypeScript “我确定这个值不是 null 或 undefined”

跳过类型检查：强制编译器忽略可能的空值警告
运行时不做任何事情：只是编译时的类型断言

为什么之前需要 !：

API 返回类型是 string | null
即使有 if (data) 检查，TypeScript 有时仍认为可能是 null
使用 ! 强制告诉编译器这里不会是 null
